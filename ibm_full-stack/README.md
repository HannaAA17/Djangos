# IBM Full Stack Track Projects

This directory contains projects completed as part of the IBM Full Stack Software Developer learning path.

## What Is Unique About This Project Set

- **True full-stack breadth**: Includes backend Django work, Node/Express services, React front-end apps, and a capstone implementation.
- **Technology transition practice**: Demonstrates moving between Python-based and JavaScript-based stacks while keeping similar product goals.
- **Lab-to-capstone progression**: Starts with targeted labs (`ibm-express`, `ibm-react`, `ibm-django`) and extends into integrated capstone deliverables.

## Structure

- `ibm-django/` — Django exercises and app structure practice.
- `ibm-express/` — Node.js and Express labs, including HTTP server and auth-focused work.
- `ibm-react/` — React practice apps and Redux-oriented examples.
- `ibm-capstone/` — End-to-end capstone artifacts with server and frontend sections.

## Professional Summary

This folder showcases platform versatility: the same engineering fundamentals (routing, CRUD, auth, data modeling, and UI state) are applied across multiple frameworks. It is especially useful as evidence of adaptability when moving between backend and frontend roles.
