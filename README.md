# Djangos Repository Portfolio

This repository is a multi-track learning portfolio that documents hands-on work across Django, Django REST APIs, full-stack JavaScript, and capstone projects. It combines coursework from multiple providers into one place so you can compare approaches, patterns, and stack choices side by side.

## Professional Overview

The repo is organized as a set of project families:

- **`ibm_full-stack/`**: IBM Full Stack Software Developer work covering Django, Express/Node.js, React, and a capstone app.
- **`meta_django/`**: Meta Back-End Developer specialization projects focused on Django fundamentals, APIs, and full-stack integration.
- **`lynda_django/`**: A LinkedIn Learning Django Essentials implementation centered on a clean notes application.
- **`dj4e_projects/`**: Django for Everybody exercises and mini-projects, including CRUD, templates, and deployment-style structure.

## What Makes This Repository Unique

1. **Cross-curriculum perspective**: Multiple learning paths are preserved together, making it easier to compare teaching styles and implementation decisions.
2. **Polyglot full-stack exposure**: Includes both Python/Django and JavaScript/Node/React ecosystems in one portfolio.
3. **Progressive complexity**: Ranges from focused labs and feature exercises to capstone-level project assembly.
4. **Reusable reference value**: The structure works as a personal knowledge base for common Django and full-stack patterns.

## Suggested Navigation

If you are reviewing this repository for technical growth:

1. Start with `meta_django/` for API-first Django foundations.
2. Review `lynda_django/` for clean class-based-view patterns in a smaller app.
3. Explore `dj4e_projects/` for breadth of Django concepts and exercises.
4. Finish with `ibm_full-stack/` to see cross-stack integration and capstone work.
